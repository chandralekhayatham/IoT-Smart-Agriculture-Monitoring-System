# IoT-Enabled Smart Agriculture Monitoring System

## Overview

The IoT-Enabled Smart Agriculture Monitoring System is a Python-based IoT simulation that monitors soil moisture, temperature, humidity, and light intensity. Based on sensor readings, it automatically controls irrigation and generates alerts to improve crop health and reduce water wastage.

---

## Features

- Soil Moisture Monitoring
- Temperature Monitoring
- Humidity Monitoring
- Light Intensity Monitoring
- Automatic Irrigation Control
- Alert Generation
- CSV Data Logging
- Python-Based IoT Simulation

---

## Technologies Used

- Python
- IoT Concepts
- Sensor Simulation
- CSV File Handling

---

## How to Run

```bash
pip install pandas matplotlib
python main.py
```
